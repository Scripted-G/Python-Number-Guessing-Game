## About

This is a web-based version of a classic number guessing game. The computer picks a random number between 1 and 100, and you try to guess it. After each guess, you'll get feedback on whether your guess was too high or too low.

## Play Online

[Play the game here!](https://python-number-guessing-game-sh6mcqmbqpqnwxd94iucga.streamlit.app/)

## Features

- Random number generation between 1-100
- Real-time feedback on your guesses
- Tracks the number of attempts
- Clean, simple interface
- Play again option

## Run Locally

If you want to run this on your own machine:

1. Clone the repository:
```bash
git clone https://github.com/Scripted-G/Python-Number-Guessing-Game.git
cd Python-Number-Guessing-Game
```

2. Install requirements:
```bash
pip install -r requirements.txt
```

3. Run the app:
```bash
streamlit run app.py
```
